# **💡 Basic Components**

**LEDs (Red: 5, Yellow: 5, Blue: 5, RGB: 1)**

EX: ໃຊ້ເປັນຕົວບົ່ງບອກສະຖານະ - ເຊັ່ນ: LED ສີແດງຕິດເມື່ອອຸນຫະພູມສູງເກີນໄປ, LED RGB ປ່ຽນສີຕາມໂໝດຕ່າງໆ.

[![LED (Light-emitting diode) explained - Soldered Electronics](https://soldered.com/productdata/2023/03/led-5mm-green-wikipedia.jpg)

**RGB module**

EX: ໃຊ້ສ້າງແສງສີຕ່າງໆ ເພື່ອຕົກແຕ່ງ ຫຼື ສະແດງສະຖານະທີ່ສັບຊ້ອນ, ເຊັ່ນ: ປ່ຽນສີຕາມອຸນຫະພູມຫ້ອງ.

![[Pasted image 20251024103841.png]]

**Resistors (220Ω, 1kΩ, 10kΩ)**

EX: ຕ້ານ 220Ω ໃຊ້ປົກປ້ອງ LED ບໍ່ໃຫ້ໄຟຫຼາຍເກີນໄປ, ຕ້ານ 10kΩ ໃຊ້ເປັນ pull-down resistor ກັບປຸ່ມກົດ.

![[Pasted image 20251024104251.png]]

![[Pasted image 20251024104401.png]]

**Push Buttons (x4 with Lids )**

EX: ໃຊ້ເປັນປຸ່ມຄວບຄຸມ - ກົດເພື່ອເປີດ/ປິດ LED, ສັບໂໝດ ຫຼື ເພີ່ມ/ລົດຄ່າຕົວນັບ.

![[Pasted image 20251024145501.png]]

**Potentiometer (5kΩ)**

EX: ໃຊ້ປັບຄວາມສະຫວ່າງຂອງ LED, ປັບຄວາມໄວມໍເຕີ ຫຼື ຄວບຄຸມສຽງຂອງ buzzer.

![[Pasted image 20251024145844.png]]

# **📟 Displays & Output**

**16x2 LCD display**

EX: ສະແດງຂໍ້ຄວາມ ເຊັ່ນ: "ອຸນຫະພູມ: 28°C", "ຍິນດີຕ້ອນຮັບ" ຫຼື ຂໍ້ມູນຈາກເຊັນເຊີຕ່າງໆ.

![[Pasted image 20251024151456.png]]

 **I2C Serial Adapter board module**

EX: ຕິດກັບ LCD 16x2 ເພື່ອຫຼຸດສາຍເຊື່ອມຕໍ່ຈາກ 16 ເສັ້ນ ເຫຼືອພຽງ 4 ເສັ້ນ (SDA, SCL, VCC, GND).

![[Pasted image 20251024151707.png]]

 **7-segment display (Common Cathode +)**

EX: ສະແດງຕົວເລກ 0-9 ເພື່ອນັບເວລາ, ສະແດງຄະແນນເກມ ຫຼື ອຸນຫະພູມ.

![[Pasted image 20251024151903.png]]

**4-Digit 7-Segment Display**

EX: ຈໍ 4 ຫຼັກໃຊ້ເທັກນິກ "ສະຫຼັບໄວໆ" ໂດຍຕິດພຽງ 1 ຫຼັກຕໍ່ຄັ້ງແບບໄວມາກ (ປະມານ 1-5 milliseconds) ເຮັດໃຫ້ຕາຄົນເຫັນເປັນວ່າຕິດພ້ອມກັນທັງ 4 ຫຼັກ.

![[Pasted image 20251024152537.png]]

**8x8 Dot Matrix display**

EX: ສະແດງຮູບໜ້າຍິ້ມ, ຂໍ້ຄວາມເລື່ອນ, ກາຕູນເຄື່ອນໄຫວງ່າຍໆ ຫຼື ເກມງູກິນຫາງ

![[Pasted image 20251024152735.png]]

#### 📡 Sensors & Input Modules

**Temperature and humidity sensor (DHT11)**

EX: ວັດອຸນຫະພູມຫ້ອງ ແລະຄວາມຊຸ່ມ ແລ້ວສະແດງຜົນຢູ່ຈໍ LCD - ເຫມາະສຳລັບລະບົບຕູ້ປູກຜັກ.

![[Pasted image 20251024153150.png]]

**LM35 Temperature Sensor**

EX: ວັດອຸນຫະພູມທີ່ແມ່ນຍຳ ໃຊ້ກັບໂປຣເຈກຄວບຄຸມອຸນຫະພູມ - ເປີດພັດລົມເມື່ອຮ້ອນເກີນ 30°C.

![[Pasted image 20251024153345.png]]

**Tilt sensor (x2)**

EX: **ເຊັນເຊີຄວາມເອຽງ** (Tilt Sensor) ແມ່ນອຸປະກອນທີ່ກວດຈັບການອຽງ ຫຼື ການປ່ຽນແປງທ່າທາງຂອງວັດຖຸ.

![[Pasted image 20251024153759.png]]

**Photoresistor (LDRs x3)**

EX: ກວດຈັບແສງສະຫວ່າງ - ເປີດໄຟ LED ອັດຕະໂນມັດເມື່ອມືດ, ປັບຄວາມສະຫວ່າງຈໍຕາມສະພາບແສງ.

![[Pasted image 20251024154011.png]]

**PIR sensor**

EX: ກວດຈັບການເຄື່ອນໄຫວຂອງຄົນ - ເປີດໄຟເມື່ອມີຄົນຜ່ານ, ລະບົບແຈ້ງເຕືອນໂຈນ.

![[Pasted image 20251024154548.png]]

 **Ultrasonic module**

EX: ວັດໄລຍະທາງ 2cm - 400cm - ເຮັດລະບົບຊ່ວຍຈອດລົດ, ວັດລະດັບນ້ຳໃນຖັງ.

![[Pasted image 20251025170311.png]]

 **Sound sensor**

EX: ກວດຈັບສຽງດັງ - ເປີດ/ປິດໄຟດ້ວຍການຕົບມື, ວັດລະດັບສຽງເສດ.

![[Pasted image 20251025170505.png]]

 **Water sensor**

EX: ກວດຈັບນ້ຳຮົ່ວ - ແຈ້ງເຕືອນເມື່ອຕຸກນ້ຳເຕັມ ຫຼື ກວດຈັບນ້ຳຝົນ.

![[Pasted image 20251025170736.png]]

![[Pasted image 20251025170852.png]]

**Flame sensor**

EX: ກວດຈັບແສງໄຟ ຫຼື ເປົາໄຟ - ລະບົບແຈ້ງເຕືອນໄຟໄໝ້ດ້ວຍສຽງ buzzer.

![[Pasted image 20251025184812.png]]

**RFID module**

EX: ໃຊ້ກັບບັດ RFID ເພື່ອລະບົບເປີດປະຕູ - ສະແກນບັດຖືກຕ້ອງຈຶ່ງປົດລັອກ.

![[Pasted image 20251025185103.png]]

**RFID tag**

EX: **:** ບັດສຳລັບສະແກນເຂົ້າອອກ, ລົງທະບຽນເວລາເຂົ້າວຽກ ຫຼື ລະບົບຈ່າຍເງິນ.

![[Pasted image 20251025185607.png]]

**Infrared receiver**

EX: ຮັບສັນຍານຈາກຣີໂມດ - ຄວບຄຸມໄຟ, ທີວີ ຫຼື ອຸປະກອນໄຟຟ້າຕ່າງໆ.


![[Pasted image 20251025185756.png]]

# **🎮 Remote & Control**

 **Infrared remote control**

EX: ຄວບຄຸມ Arduino ຈາກໄກ - ກົດປຸ່ມເພື່ອປ່ຽນສີ LED, ປິດເປີດອຸປະກອນ.

![[Pasted image 20251025190431.png]]

**Joystick module** 

EX: ຄວບຄຸມທິດທາງ (ຂຶ້ນ, ລົງ, ຊ້າຍ, ຂວາ) - ໃຊ້ກັບເກມ ຫຼື ຄວບຄຸມໂຣບັອດ.

![[Pasted image 20251025190715.png]]

**4x4 Matrix Keyboard Module**

EX: ປ້ອນລະຫັດຜ່ານ 4 ຕົວເລກເພື່ອປົດລັອກ, ປ້ອນເບີໂທລະສັບ ຫຼື ຄຳສັ່ງຕ່າງໆ.

![[Pasted image 20251025191315.png]]

**Relay module**

EX: ເປີດ/ປິດອຸປະກອນໄຟຟ້າ 220V - ຄວບຄຸມໂຄມໄຟ, ພັດລົມ ຫຼື ປັ້ມນ້ຳ.
